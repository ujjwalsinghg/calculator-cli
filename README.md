# 🧮 Console Calculator in Python

This is a simple command-line calculator built using Python. It performs basic arithmetic operations:

✅ Addition  
✅ Subtraction  
✅ Multiplication  
✅ Division

## 🔧 How to Run

Make sure Python is installed.

```bash
python calculator.py
```

## 📚 Skills Used
- Python functions
- Loops and conditionals
- Basic input/output
